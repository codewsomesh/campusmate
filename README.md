# CampusMate 🎓

CampusMate is a modern campus marketplace built for college students to buy, sell, rent, and discover products and services within their campus community.

## Features

- Modern responsive UI
- Browse product listings
- Detailed listing pages
- Category browsing
- Search interface
- Mobile-friendly design
- Built using Next.js App Router

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React
- React Icons

## Future Improvements

- User Authentication
- Supabase Database
- Image Uploads
- Wishlist
- Messaging System
- Advanced Search & Filters

## Getting Started

```bash
npm install
npm run dev
```
